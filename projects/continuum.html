What it does

One app. Every corner of your Mac.

Continuum is a single, self-contained native app spanning cleaning, disk visualization, security, privacy, app management, and deep hardware telemetry — so you don't need eight separate utilities (or a subscription to each one).

🧹
Cleaner
Scans whitelisted caches, logs, DerivedData and orphaned app support for reclaimable space. Previews everything and moves to Trash — never hard-deletes.
🗺️
Disk Map
An interactive treemap of any folder or volume. Click to zoom, double-click to reveal in Finder, with an optional admin-scan for root-owned files.
🛡️
Security Scanner
Malware / adware / spyware scanning across three tiers — standard, deep, forensic — with reversible quarantine, never permanent deletion.
🗑️
Smart Uninstaller
Finds every on-disk trace of an app — bundle, launch agents, app support, caches, container data — and removes only what you choose.
🔌
Extensions & Permissions
Review and toggle extensions, login items, and launch agents in one dashboard. See exactly which apps hold camera, mic, or full-disk access.
📡
Network Monitor
A reverse firewall that surfaces outbound connections in real time and blocks unauthorized hosts via a dedicated pf anchor.
🔥
Shredder & Free-Space Wipe
Multi-pass secure erase for sensitive files, plus a free-space wipe that overwrites unused blocks so deleted files can't be recovered — live data is never touched.
🌡️
Hardware & Battery
Every SMC temperature sensor, live fan RPM and custom fan curves, plus battery health, cycle count, and live charge/discharge wattage.
📊
Drive Health & Sensors
NVMe S.M.A.R.T. data and lifetime terabytes written, plus ambient light, lid angle and accelerometer readings with scriptable automations.
Honest about its limits: Not every Mac has every sensor, and some hardware has no stable public API — Continuum says so in the UI instead of faking a reading. The network monitor has no kernel filter entitlement, so it watches established connections and enforces blocks via pf rather than claiming packet-level interception.
